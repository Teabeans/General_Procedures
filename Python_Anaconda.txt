Purpose: To install the Python programming language
         To install the Anaconda (Python and R distribution, plus package management)
         
I. Download the Anaconda installer
   A. https://www.anaconda.com/download/

II. Follow instructions for installation
   A. https://conda.io/docs/user-guide/install/windows.html
   B. Accept EULA
   C. Select "Just Me"
   D. Choose installation folder
   E. Set Anaconda as the default Python
   F. (Optional) Install VS Code for use as your Python IDE
   G. Finish

III. Open Anaconda

IV. If using VSCode as your IDE
   A. Open the Extensions manager (left side of screen)
   B. Search for and locate the GitHub manager
   C. Install
   D. Restart
   E. Create a PAT (Personal Access Token) - aka. login credentials
      1. On GitHub - Verify email address
      2. On GitHub - Go to "Settings" (upper right, profile picture)
      3. On GitHub - On left sidebar => Developer Settings => Personal Access Tokens
      4. "Generate New Access Token"
      5. Enter password to confirm
      6. Complete prompts
